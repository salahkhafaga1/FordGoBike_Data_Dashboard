#  Ford GoBike Data Analysis & Interactive Dashboard

##  Project Overview
This project is an end-to-end data analysis pipeline for the **Ford GoBike System (2019)**. It starts with raw data assessment and cleaning, moves through exploratory data analysis (EDA), and culminates in a fully interactive, multi-page web application built with **Python, Plotly, and Dash**.

##  Features
- **Multi-page Architecture:** Built using Dash Pages.
- **Interactive Visualizations:** Powered by Plotly (Pie charts, Histograms, Boxplots).
- **Responsive UI:** Styled with Dash Bootstrap Components.

##  Technology Stack
- Python, Pandas, Numpy, Plotly, Dash.

##  How to Run Locally
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Add the cleaned CSV file to the `data/` folder.
4. Run the application: `python app.py`
